  3. L2-Norm

    Fri 08 January 2016
    By Lukas Woodtli

    L2-Norm

    A vector \(\vec a\) has the \(L_2\)-Norm:

    $$\left \| \vec a \right \|_{2} = \sqrt{\sum_{k=1}^n|a_k|^2}$$

    \(n\): Dimensions \(a_k\): \(k\)-th element of \(a\)

    L2-Distance

    The \(L_2\)-Distance between two vectors \(\vec a\) and \(\vec b\) is:

    $$d \left ( \vec a, \vec b \right ) = \sqrt{\sum_{k=1}^n|a_k - b_k|^2}$$ ...

    Distant Functions

    A distant function works on two elements (i.e. vectors, sets...).

    A given distance function \(d(s,t)\) needs to fulfill properties:

    1. \(d(s,t) \geq 0\) (non-negative)
    2. \(d(s,s) = 0\) (distance to itself is zero)
    3. \(d(s,t) = d(t,s)\) (symmetric)
    4. \(d(s,t) + d(t,r) \geq d(s,r)\) ...
