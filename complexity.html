Complexity

Sun 08 January 2017

This page is work in progress

“Halving the size of the problem at each step is the distinguishing characteristic of logarithmic growth”

SICP: 2.3.3 Example: Representing Sets

Main complexity names:

Running time Name
O(1) constant
O(n) linear
O(log(n)) logarithmic
O(n*log(n)) linearithmic
O(n^2) quadratic
O(2^n) exponential

Algorithm with exponential run time are considered as not feasible.

Amortized time means that there is no guarantee that the algorithm is run every time with that complexy but that an average value of the run time is taken.
